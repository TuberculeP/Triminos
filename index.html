Triminos

Comment utiliser ?

Le dessin est lancé par la fonction problemeFinal(n) avec n le niveau de récursion (un naturel supérieur à 0).
Un éditeur de code est disponible à l'icone de crayon sur l'interface Trinket pour changer de récursion ainsi que de modifier le code côté client si vous le souhaitez.

Plus la récurrence est grande, plus Turtle prendra du temps à le réaliser.

En prenant le temps d'apréhender le code, vous pouvez également via la commande problemeFinal(n, trou) indiquer les coordonnées du pixel de départ : le trou est une paire au format (x,y) avec (0,0) le coin inférieur gauche. le trou peut être n'importe quelle paire entre (0,0) et (2n -1, 2n -1).
